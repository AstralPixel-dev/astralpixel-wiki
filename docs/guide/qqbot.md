# QQ群机器人使用方法
| 指令                                                          | 简介                         |
| ------------------------------------------------------------- | ---------------------------- |
| 注销+ID | 解绑白名单                         |
| 查询+@玩家/输入玩家ID| 查询玩家信息                         |
| mcs  | 查询服务器状态
| 我的信息  | 查询自己账号状态
| 帮助/help | 查看机器人命令
