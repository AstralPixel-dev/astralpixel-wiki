# AstralPixel 老玩家认证规则

## 🌟 认证标准
### 基础认证
- **游戏时长** ≥ 168小时  
- **账号要求**：  
  - 无任何违规记录  
  - 非小号  

### 高阶认证
- **游戏时长** ≥ 240小时  
- **账号要求**：  
  - 无任何违规记录  
  - 非小号   
  - 时长非挂机所得
  - 在服务器内有一定建筑/贡献

|------------|----------------|
